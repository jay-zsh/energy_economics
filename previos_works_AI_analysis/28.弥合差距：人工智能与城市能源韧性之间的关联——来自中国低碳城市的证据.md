## 关键词

## 摘要

低碳城市的发展对于实现可持续、高质量增长至关重要，并在确保能源生产和分配网络的可靠性和稳定性方面发挥关键作用。本研究采用PSM-DID模型、双重机器学习模型和空间杜宾模型，实证分析了低碳城市试点对能源韧性的影响、潜在机制和空间效应。基准回归结果表明，低碳城市试点显著增强了城市能源韧性，但在不同类型的城市中表现出异质性结果。在大型城市（人口超过100万）和人口净流入城市中，低碳城市试点能大幅提升城市能源韧性，而在小型城市（人口少于100万）和人口净流出城市中，低碳城市建设对城市能源韧性产生负面影响。机制分析表明，人工智能的应用是低碳城市试点推动城市能源韧性提升的主要渠道，但效应呈现非线性增长。当人工智能应用水平超过4.2909时，低碳城市试点对城市能源韧性的增强作用将提升。此外，空间分析显示，低碳城市试点不仅增强了试点城市的能源韧性，还对周边城市产生溢出效应。本文提出的一系列方法不仅检验了低碳城市试点与城市能源韧性之间的因果关系，还为提升城市能源系统韧性提供了新的经验和证据。

## 研究思路

### 为什么要做这个研究

#### 现实重要性

论文指出，中国面临极端天气事件频发、地缘政治紧张加剧和国际能源市场波动等多重挑战，导致能源进口渠道和运输路线受限，能源供应链中断风险上升。中国作为全球最大的能源消费国之一，其快速经济增长和工业化进程推动了能源需求的持续增长，例如2023年能源进口量同比增长27.2%。这些因素增加了城市能源韧性能力的迫切需求，如果能源韧性问题未解决，可能导致能源供应不稳定、经济受损和环境恶化，影响城市可持续发展和“30·60”双碳目标的实现。理解中国城市能源韧性对于减少城市能源使用和优化能源消费模式至关重要。

#### 政策重要性

论文提到，中国国家发改委于2010年推出了低碳城市试点政策，旨在探索工业污染减少机制，推动城市绿色可持续发展和高质量能源转型。该政策已分三批建立了77个低碳城市，作为应对环境不确定性的气候政策，研究其对能源韧性的影响具有重要政策意义。研究结果可为政策制定者提供科学依据，评估试点项目有效性，优化能源韧性城市发展策略，并促进区域协同政策制定，例如通过空间溢出效应实现跨城市合作。

### 1.2 数据预处理流程

每个步骤以原子级分解，确保可操作和可验证：

- •
    
    **步骤1: 数据收集与整合**
    
    - •
        
        操作目的：将多源数据合并为统一面板数据集。
        
    - •
        
        具体操作：从各来源下载城市级年度数据，按城市代码和年份对齐；使用唯一标识符（城市ID和年份）进行数据连接。
        
    - •
        
        输入：原始分年分城市数据文件。
        
    - •
        
        输出：初步合并的数据集，包含所有变量原始值。
        
    - •
        
        验证方法：检查城市数量和年份范围是否匹配265城市×12年；使用描述性统计（如计数、缺失值比例）验证完整性。
        
    - •
        
        预期结果：数据集无重复记录，城市-年份组合完整。
        
    
- •
    
    **步骤2: 缺失值处理**
    
    - •
        
        操作目的：确保数据连续性，避免偏差。
        
    - •
        
        具体操作：对缺失变量值使用线性插值法填充；对于城市-年份缺失，优先使用相邻年份数据插值。
        
    - •
        
        输入：初步合并数据集。
        
    - •
        
        输出：缺失值处理后的数据集。
        
    - •
        
        验证方法：计算插值前后缺失值比例，确保降至0%；对比插值值与原始数据分布，检查异常。
        
    - •
        
        预期结果：所有变量无缺失，插值后数据分布合理（如均值、标准差变化小）。
        
    
- •
    
    **步骤3: 变量计算与转换**
    
    - •
        
        操作目的：生成核心变量，如能源韧性(ER)和人工智能(AI)。
        
    - •
        
        具体操作：
        
        - •
            
            **核心变量测算：能源韧性(ER)**：
            
            - •
                
                测算公式：使用熵值法计算综合指数，基于论文表1的指标系统。公式为：首先标准化各指标，然后计算熵值 ej​=−lnn1​∑i=1n​pij​lnpij​，其中 pij​为指标j在城市i的标准化值，n为城市数；权重 wj​=∑j​(1−ej​)1−ej​​；最终ER = ∑(w_j × 标准化指标值)。
                
            - •
                
                数据来源：指标值来自统计年鉴，如天然气供应量、绿化覆盖率等。
                
            - •
                
                计算步骤：收集6个指标原始数据→标准化处理→计算熵值和权重→加权求和得ER。
                
            - •
                
                示例计算：以某城市为例，假设指标值标准化后为[0.5, 0.6, ...]，权重为[0.3057, 0.3137, ...]，则ER = 0.5×0.3057 + 0.6×0.3137 + ... ≈ 0.030（匹配表3均值）。
                
            - •
                
                变量含义：ER表示城市能源系统应对冲击的能力，值越高韧性越强。
                
            
        - •
            
            **核心变量测算：人工智能(AI)**：
            
            - •
                
                测算公式：AI = ∑(L_ijt / L_it) × (Rob_jt / L_jt)，其中L_ijt为城市i行业j的就业，L_it为城市总就业，Rob_jt为全国行业j的机器人存量，L_jt为全国行业j就业。
                
            - •
                
                数据来源：机器人数据来自国际机器人联合会(IFR)，就业数据来自中国第二次经济普查和统计年鉴。
                
            - •
                
                计算步骤：对齐行业分类→计算各城市行业就业比例→乘以全国机器人密度→求和得AI指数。
                
            - •
                
                示例计算：假设城市i的制造业就业占比0.3，全国制造业机器人密度0.5，则贡献为0.3×0.5=0.15；重复所有行业求和。
                
            - •
                
                变量含义：AI表示机器人应用密度，衡量AI技术渗透水平。
                
            
        
    - •
        
        输入：原始变量数据。
        
    - •
        
        输出：计算后的ER、AI及控制变量（如取对数处理Pergdp）。
        
    - •
        
        验证方法：检查ER值范围[0.004, 0.625]和AI值范围[0, 25.911]是否匹配表3；通过相关性分析验证指标合理性。
        
    - •
        
        预期结果：变量计算正确，分布与论文一致。
        
    
- •
    
    **步骤4: 数据清理**
    
    - •
        
        操作目的：处理异常值和重复数据。
        
    - •
        
        具体操作：使用箱线图识别异常值（如ER>0.625或AI>25.911视为异常），采用缩尾处理（1%和99%分位数替换）；删除完全重复行。
        
    - •
        
        输入：变量计算后数据集。
        
    - •
        
        输出：清洁数据集。
        
    - •
        
        验证方法：描述性统计检查异常值比例降至0%；唯一性检查确保无重复。
        
    - •
        
        预期结果：数据质量高，适合模型输入。

### 2.2 理论方法与公式

- •
    
    **方法概述**：论文使用PSM-DID评估政策效应，双重机器学习处理内生性，空间杜宾模型分析空间效应。
    
- •
    
    **数学公式**：
    
    - •
        
        **DID模型**（公式1）：
        
        ERi,t​=β0​+β1​LCCTi,t​+γControli,t​+λi​+μt​+εi,t​
        
        符号含义：i为城市，t为年份，LCCT为政策虚拟变量，Control为控制变量集，λ和μ为固定效应。
        
    - •
        
        **AI中介模型**（公式4-6）：
        
        AIi,t​=α0​+α2​LCCTi,t​+γControli,t​+λi​+μt​+εi,t​
        
        ERi,t​=α0​+α3​LCCTi,t​+α4​AIi,t​+γControli,t​+λi​+μt​+εi,t​
        
    - •
        
        **空间杜宾模型**（公式10）：
        
        ERit​=ρWERit​+α1​LCCTit​+∑βXit​+θW(LCCTit​+∑Xit​)+γt​+ui​+εit​
        
        其中W为空间权重矩阵。
        
    
- •
    
    **方法假设**：DID需满足平行趋势假设；空间模型假设空间依赖性。
    
- •
    
    **选择理由**：PSM-DID处理选择偏差，双重机器学习解决高维数据问题，空间模型捕捉溢出效应。

### 1.2 方法局限性分析

- •
    
    **PSM匹配**：
    
    - •
        
        局限性：依赖可观测变量，无法处理不可观测混杂；匹配质量受变量选择影响。
        
    - •
        
        影响：可能导致残余偏差，影响因果推断可靠性。
        
    - •
        
        AI解决可能性：使用机器学习自动学习倾向得分，减少人为选择偏差。
        
    
- •
    
    **DID回归**：
    
    - •
        
        局限性：平行趋势假设可能不成立；线性模型无法捕捉复杂非线性关系。
        
    - •
        
        影响：政策效应估计可能有偏。
        
    - •
        
        AI解决可能性：用神经网络捕捉非线性，或强化学习优化模型设定。
        
    
- •
    
    **AI测量**：
    
    - •
        
        局限性：AI代理变量（机器人密度）粗糙，无法捕捉AI全貌（如算法效率、自然语言处理等）。
        
    - •
        
        影响：中介效应估计可能低估。
        
    - •
        
        AI解决可能性：直接使用AI应用数据（如专利、API调用）或深度学习提取特征。
        
    
- •
    
    **阈值模型**：
    
    - •
        
        局限性：单门槛假设简单，可能忽略多阈值或连续变化。
        
    - •
        
        影响：非线性效应刻画不充分。
        
    - •
        
        AI解决可能性：用递归神经网络或变分自编码器学习动态阈值。
        
    
- •
    
    **空间模型**：
    
    - •
        
        局限性：权重矩阵预设（经济/地理），无法学习自适应空间依赖。
        
    - •
        
        影响：空间效应估计可能不精确。
        
    - •
        
        AI解决可能性：图神经网络自动学习空间关系。
        
    
- •
    
    **双重机器学习**：
    
    - •
        
        局限性：依赖基学习器选择（如随机森林），可解释性差。
        
    - •
        
        影响：结果解释困难。
        
    - •
        
        AI解决可能性：用可解释AI（如SHAP）增强透明度。

# 研究-定题

AI驱动的低碳城市试点对城市能源韧性影响评估：基于多模态深度学习与空间图神经网络的实证研究

## 研究内容

### 1. 研究目的与意义

本研究旨在通过人工智能方法优化低碳城市试点（LCCT）政策对城市能源韧性（ER）的评估框架。现实背景方面，全球能源危机、气候变化加剧及中国"双碳"目标下，城市能源系统韧性成为可持续发展核心议题。政策层面，中国已实施三批低碳城市试点，但传统评估方法存在线性假设局限和测量偏差。AI优化后，本研究理论贡献在于构建AI赋能的能源韧性评估新范式，突破传统计量经济学假设约束；实务贡献在于为智慧城市政策提供精准决策支持，助力能源安全战略实施。通过集成多模态数据和先进AI算法，研究将提升政策效应估计的准确性和可解释性，为全球城市能源转型提供参考。

### 2. 研究问题与假设

核心研究问题：如何利用多模态深度学习与空间图神经网络方法，更精确地评估低碳城市试点政策对城市能源韧性的因果效应及空间溢出机制？

研究假设：

- •
    
    H1：AI优化后的评估模型能显著提升政策效应估计精度，揭示非线性阈值效应（如AI应用水平>4.2909时效应增强）。
    
- •
    
    H2：空间图神经网络可自适应识别低碳城市试点的跨区域溢出效应，优于预设权重矩阵方法。
    
- •
    
    H3：多模态数据（政策文本、卫星图像）融合能更直接度量AI应用水平，减少代理变量偏差。
    

### 3. 研究方法与技术路线

本研究继承原论文的PSM-DID基础框架，但全面引入AI方法优化关键环节：

- •
    
    **数据准备**：收集265个中国城市2011-2022年的多源数据，包括结构化统计数据（如能源消耗、经济指标）、非结构化政策文本（政府工作报告）及卫星图像（夜间灯光、绿化覆盖）。数据预处理采用自然语言处理（BERT模型）提取政策AI关键词嵌入，卷积神经网络（CNN）预处理图像特征。
    
- •
    
    **AI方法介入环节**：
    
    1. 1.
        
        **倾向得分匹配优化**：用XGBoost替代传统logit模型，自动学习匹配变量重要性，提升平衡性。
        
    2. 2.
        
        **政策效应估计**：使用Transformer架构建模时间序列依赖，捕捉LCCT的动态非线性效应；对比原DID模型，加入注意力机制识别异质性处理效应。
        
    3. 3.
        
        **AI测量革新**：基于BERT的政策文本分析生成直接AI指数，替代机器人密度代理变量，增强中介效应检验可靠性。
        
    4. 4.
        
        **空间分析增强**：采用图神经网络（GNN）构建城市网络，自适应学习空间依赖关系，替代预设经济/地理权重矩阵。
        
    
- •
    
    **评估指标**：通过均方根误差（RMSE）、R²对比模型性能；使用SHAP值进行可解释性分析；通过Bootstrap验证稳健性。
    

### 4. 预期结果与创新点

预期主要结论：

- •
    
    AI模型将揭示LCCT对ER的非线性促进效应，阈值分析显示AI应用水平超过4.5时政策效果倍增。
    
- •
    
    空间溢出效应显著，GNN识别出核心城市对周边区域的辐射半径达150公里。
    
- •
    
    多模态数据融合提升AI测量效度，中介效应系数增幅达20%。
    
    创新点：
    
- •
    
    **理论创新**：提出"AI-能源韧性"评估框架，整合复杂系统理论与机器学习，突破线性范式。
    
- •
    
    **方法创新**：首创多模态深度学习与GNN结合的空间政策评估模型，解决传统方法预设偏差问题。
    
- •
    
    **应用创新**：为低碳城市政策提供动态优化工具，支持精准靶向干预。
    
    局限与对策：数据获取成本高，采用迁移学习降低标注需求；模型可解释性挑战，集成SHAP工具增强透明度。
    

## 摘要

在全球能源转型与气候变化背景下，城市能源韧性成为可持续发展的关键。本研究针对低碳城市试点政策评估的局限性，引入人工智能方法优化评估框架。通过整合多模态数据（统计指标、政策文本和卫星图像），采用XGBoost、Transformer架构及图神经网络，重构倾向得分匹配、政策效应估计和空间分析环节。预期结果表明，AI模型能显著提升评估精度，揭示非线性阈值效应（如AI应用水平>4.2909时政策效果增强）和跨区域溢出机制。理论层面，本研究构建了AI赋能的能源韧性评估新范式；实践层面，为智慧城市政策提供精准决策支持，推动能源安全战略实施。方法创新包括多模态数据融合与自适应空间学习，突破传统计量经济学假设约束，为全球城市低碳转型提供实证基础。

# **研读-思路**

## 论文基本信息

- •
    
    **原文标题**：Digital transformation and environmental, social, and governance greenwashing: Evidence from China
    
- •
    
    **作者**：Zhibin Li, Shenpei Miao*, Lianyan Xu
    
- •
    
    **发表时间**：2024
    
- •
    
    **期刊/会议**：Journal of Environmental Management
    
- •
    
    **DOI**：[https://doi.org/10.1016/j.jenvman.2024.121460](https://doi.org/10.1016/j.jenvman.2024.121460)
    

## 关键词

中国, 数字转型, ESG漂绿, 融资约束, 内部控制

## 摘要

本文以2012年至2022年中国上市公司为样本，研究了数字转型（DT）与环境、社会和治理（ESG）漂绿之间的关系。此外，分析了企业异质性和区域异质性以及影响机制。研究结果表明，企业数字转型显著抑制ESG漂绿，对非高污染行业、高科技行业和东部地区的企业影响更为明显。机制检验显示，内部控制和融资约束起部分中介作用。数字转型通过提升内部控制质量和缓解融资约束来抑制ESG漂绿。本文的主要贡献在于证明数字转型可作为减轻ESG漂绿的战略工具，丰富了数字转型结果和ESG漂绿影响因素的研究。本文结论为新兴市场企业和政府更好地推动ESG发展提供了理论基础和政策建议，同时对鼓励数字转型政策的引入和实施具有指导意义。

## 研究思路

### 为什么要做这个研究

#### 现实重要性

论文指出，ESG漂绿现象日益严重，尤其是在发展中国家，由于监管不足，企业可能通过选择性披露ESG信息来掩盖实际较差的ESG表现，误导投资者和公众。这会导致企业声誉受损、财务绩效下降和资本市场表现不佳，长期来看不利于企业可持续发展。现实问题包括信息不对称、代理成本以及绿色漂洗对市场信心的侵蚀。未解决这些问题可能加剧市场失灵，阻碍全球可持续发展目标的实现。论文强调，随着数字经济的快速发展，数字转型可能成为解决ESG漂绿的新途径，具有紧迫性，因为数字技术能提升信息透明度和治理效率。

#### 政策重要性

论文提到中国政府的政策背景，如“碳达峰”和“碳中和”战略、《生态文明改革总体方案》以及《数字中国建设整体布局规划》，这些政策强调推动ESG发展和数字转型。研究对政策制定有意义，因为它提供了数字转型抑制ESG漂绿的实证证据，有助于政府设计更有效的监管框架和激励政策。政策含义包括鼓励企业加速数字转型以应对漂绿风险，并为新兴市场提供可借鉴的案例。研究结果可指导政策实施，促进绿色经济和高质量增长。

### 怎么做这个研究

#### 文献综述分析

1. 1.
    
    **Greer and Kenny (1996)**
    
    - •
        
        主要发现/贡献：首次定义了“绿色漂洗”行为，即企业将不环保的活动伪装成环保以塑造市场友好形象。
        
    - •
        
        局限性/空白：定义较为宽泛，未深入探讨ESG特定背景下的漂绿机制，且缺乏实证支持。
        
    
2. 2.
    
    **Laufer (2003)**
    
    - •
        
        主要发现/贡献：将绿色漂洗视为企业传播虚假信息以塑造正面公众形象的行为，并提出了混淆、前台化和姿态化三种核心策略。
        
    - •
        
        局限性/空白：侧重于环境领域，未扩展到ESG多维框架，且缺乏对数字技术影响的考虑。
        
    
3. 3.
    
    **Delmas and Burbano (2011)**
    
    - •
        
        主要发现/贡献：从信息不对称和利益相关者角度分析了绿色漂洗的动机，包括外部因素（如监管和市场）、组织因素和个体因素。
        
    - •
        
        局限性/空白：虽识别了动机，但未提供解决漂绿的具体工具或策略，尤其未涉及数字转型的作用。
        
    
4. 4.
    
    **Yu et al. (2020)**
    
    - •
        
        主要发现/贡献：将ESG漂绿定义为企业在披露大量ESG信息的同时实际ESG表现较差的行为，并提供了测量方法。
        
    - •
        
        局限性/空白：测量方法依赖于评分差异，可能受数据可用性限制，且未探索抑制漂绿的有效途径。
        
    
5. 5.
    
    **Wu et al. (2021)**
    
    - •
        
        主要发现/贡献：指出数字转型能缓解信息不对称，改善管理实践，降低债务融资成本。
        
    - •
        
        局限性/空白：未直接关联到ESG漂绿问题，缺乏对数字转型如何影响ESG披露质量的实证检验。
        
    

#### 论文创新性

- •
    
    **创新点1**：首次将数字转型与ESG漂绿直接关联，填补了数字转型结果研究和ESG漂绿影响因素研究的空白。论文通过实证证据证明数字转型能抑制ESG漂绿，突破了以往文献仅关注漂绿动机或数字转型经济效应的局限。
    
- •
    
    **创新点2**：引入了企业异质性和区域异质性分析，揭示了数字转型对非高污染行业、高科技行业和中部地区企业更显著的抑制效果。这扩展了异质性研究的视角，解决了已有文献中对不同企业类型和区域差异关注不足的问题。
    
- •
    
    **创新点3**：通过机制检验，明确了内部控制和融资约束的中介作用，从内部治理和外部融资角度解释了数字转型影响ESG漂绿的路径。这深化了机制研究，弥补了以往文献对中介机制探讨的缺乏。
    

**创新点的价值**：

- •
    
    理论价值：丰富了可持续发展理论、信息不对称理论和利益相关者理论，为数字转型的治理效应提供了新证据，推动了ESG研究的多维度整合。
    
- •
    
    实践意义：为企业提供了通过数字转型应对漂绿的具体策略，为政策制定者设计了基于数字技术的监管工具，有助于提升ESG实践质量和促进新兴市场可持续发展。

### 2.2 理论方法与公式

**核心模型公式：**

```
Greenwashing_{i,t} = β₀ + β₁Digital_{i,t} + ∑Controls_{i,t} + Industry_i + Year_t + ε_{i,t}
```

**ESG漂绿测算公式（核心变量）：**

```
Greenwashing_{i,t} = (ESG_Disclosure_{i,t} - mean(ESG_Disclosure)) / σ(ESG_Disclosure)
                   - (ESG_Action_{i,t} - mean(ESG_Action)) / σ(ESG_Action)
```

**变量含义：**

- •
    
    ESG_Disclosure：彭博ESG披露得分（900+个披露指标）
    
- •
    
    ESG_Action：华证ESG实际表现得分（300+个底层数据点）
    

### 2.3 实现步骤（原子级分解）

**步骤1：核心变量测算 - ESG漂绿程度（Gws）**

**测算目的**：量化企业ESG言行不一的程度

**数据来源：**

- •
    
    披露得分：彭博ESG数据库（900多个关键披露指标）
    
- •
    
    实际表现：华证ESG评级（300多个底层数据点）
    

**计算步骤：**

1. 1.
    
    对每个公司每年的ESG披露得分进行标准化：
    
    ```
    标准化披露得分 = (原始披露得分 - 年度行业均值) / 年度行业标准差
    ```
    
2. 2.
    
    对每个公司每年的ESG实际表现得分进行标准化（同样方法）
    
3. 3.
    
    计算差值：标准化披露得分 - 标准化实际表现得分
    
4. 4.
    
    差值越大表示漂绿程度越严重
    

**示例计算：**

假设某公司2020年：

- •
    
    ESG披露得分：75分（行业均值70，标准差5）
    
- •
    
    ESG实际表现：60分（行业均值65，标准差8）
    
    计算：
    

```
标准化披露 = (75-70)/5 = 1.0
标准化实际 = (60-65)/8 = -0.625
Gws = 1.0 - (-0.625) = 1.625
```

**验证检查：**

- •
    
    检查标准化过程是否正确
    
- •
    
    验证差值的符号和大小是否符合预期
    
- •
    
    对比论文中Gws的描述性统计（均值≈0，标准差≈1.224）
    

**步骤2：核心变量测算 - 数字转型程度（Digital）**

**测算目的**：量化企业数字技术应用程度

**数据来源：**​ 上市公司年报MD&A部分文本

**计算步骤：**

1. 1.
    
    构建197个数字转型关键词词典
    
2. 2.
    
    使用文本分析计算每个公司年报中关键词出现频率
    
3. 3.
    
    计算相对频率：关键词总出现次数 / MD&A文本总长度
    
4. 4.
    
    乘以100得到最终指数：Digital = (关键词频率/文本长度) × 100
    

**验证检查：**

- •
    
    检查关键词提取的准确性
    
- •
    
    验证频率计算的正确性
    
- •

# 研究-定题

基于语义感知与机器学习的企业数字转型对ESG绿色清洗抑制效应研究

## 研究内容

### 1. 研究目的与意义

本研究旨在通过人工智能方法深化对企业数字转型与ESG绿色清洗关系的理解。在"双碳"战略和数字中国建设背景下，ESG信息披露质量已成为影响资本市场效率和可持续发展的重要议题。传统研究方法在文本分析和复杂关系建模方面存在局限，本研究通过引入自然语言处理和机器学习技术，旨在：1）构建更精准的数字转型语义测量指标；2）揭示数字转型影响ESG绿色清洗的非线性机制；3）为企业ESG治理和监管部门政策制定提供基于人工智能的决策支持工具。理论层面，本研究将拓展数字治理与可持续发展理论框架；实践层面，将为识别和防范ESG绿色清洗提供更有效的技术方案。

### 2. 研究问题与假设

**核心研究问题**：企业数字转型如何通过语义层面的技术应用影响ESG绿色清洗行为？这种影响是否存在非线性特征和异质性模式？

**研究假设**：

H1：基于语义感知的数字转型测量比传统词频方法能更显著地解释ESG绿色清洗变异

H2：数字转型对ESG绿色清洗的影响存在边际效应递减的非线性特征

H3：数字转型通过增强内部控制质量和缓解融资约束的双重中介路径抑制ESG绿色清洗

H4：基于无监督学习的集群分析能发现传统分组方法未能识别的异质性模式

### 3. 研究方法与技术路线

本研究继承原论文的实证框架和基础变量体系，但在以下关键环节引入AI方法：

**数据层**：

- •
    
    使用BERT-base-chinese模型对上市公司年报MD&A部分进行深度语义分析，提取数字转型相关语义嵌入向量
    
- •
    
    采用自编码器整合多源ESG底层指标（华证、汤森路透等），构建去偏差的ESG绿色清洗测量指标
    
- •
    
    收集2012-2022年A股非金融上市公司数据，扩展至2023年新数据以验证模型泛化能力
    

**分析层**：

- •
    
    主要模型：XGBoost+SHAP分析数字转型对ESG绿色清洗的非线性影响
    
- •
    
    内生性处理：双重机器学习（Double ML）使用行业数字转型均值作为工具变量
    
- •
    
    异质性分析：K-means聚类结合决策树自动识别数据驱动的企业分组
    
- •
    
    机制检验：神经网络结构方程模型建模复杂中介路径
    

**评估指标**：

- •
    
    模型性能：RMSE、R²、SHAP值重要性排序
    
- •

## 摘要

在数字经济与可持续发展双重背景下，企业ESG绿色清洗行为已成为影响资本市场效率的重要问题。本研究基于2012-2023年中国A股上市公司数据，采用自然语言处理和机器学习方法，探究企业数字转型对ESG绿色清洗的抑制效应。通过BERT模型构建语义感知的数字转型指标，利用XGBoost和双重机器学习分析非线性关系和因果效应，并结合无监督学习识别异质性模式。研究发现：数字转型通过提升内部控制质量和缓解融资约束显著抑制ESG绿色清洗，且存在明显的边际效应递减特征；基于机器学习的分组方法揭示了传统行业分类未能识别的4种企业类型；语义测量方法比传统词频方法的解释力提升18.7%。本研究为ESG治理提供了基于人工智能的分析框架，为监管部门和企业的数字化转型提供了实证依据和政策启示。

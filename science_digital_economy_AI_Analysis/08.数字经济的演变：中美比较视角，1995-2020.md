## 关键词

数字经济, 投入-产出分析, 政策, 美国, 中国

## 摘要

本文基于1995-2020年协调的投入-产出数据，比较分析了中美数字经济的演变，突出收敛性和持久不对称性。通过分解数字增加值为ICT生产、后向关联和非ICT部门的数字化转型，发现中国数字经济规模增长近九倍，是美国增速的四倍，相对规模从不足美国的20%升至近80%。但收敛不均衡：中国在ICT硬件和电信领域超越美国，得益于全球价值链整合、FDI和产业政策，但在软件、IT服务和数字内容领域仍显著落后，美国保持决定性优势。论文引入“追赶速度”指标，揭示中国基础设施驱动模式与美国创新驱动模式的结构性分歧，加剧战略竞争和数字贸易冲突风险。

## 研究思路

### 为什么要做这个研究

#### 现实重要性

数字技术在过去三十年中深刻重塑经济和社会系统，准确测量数字经济的规模、结构和动态对于设计包容性和前瞻性政策框架至关重要。中美作为全球两大数字经济体，其演变路径对全球数字转型具有战略影响。未解决数字经济的测量不一致性和跨國可比性问题，可能导致政策误判，影响生产力和经济韧性。

#### 政策重要性

数字经济发展涉及国家竞争力和安全，中美政策策略塑造全球数字转型方向。研究结果可为政策制定提供依据，如中国的“数字中国”和“中国制造2025”政策，以及美国的市场驱动创新策略，帮助发展中国家借鉴经验，避免数字鸿沟扩大。

## 2. 方法部分

### 2.1 软件环境与依赖

- •
    
    **软件/工具清单**：Python 3.8及以上，依赖包包括pandas、numpy用于数据处理；无特殊分析工具，I-O计算基于矩阵运算。
    
- •
    
    **环境配置**：安装Python和包 via pip，验证通过导入包和运行示例代码。
    

### 2.2 理论方法与公式

- •
    
    **方法概述**：基于Leontief投入-产出模型，分解数字经济增加值。
    
- •
    
    **数学公式**：
    
    - •
        
        总产出方程: x=(I−A)−1f
        
    - •
        
        数字经济成分:
        
        - •
            
            DE1a: ∑j∈ICT​∑i∈ICT​vj​bj,i​fi​
            
        - •
            
            DE1b: ∑j∈ICT​∑i∈/ICT​vj​bj,i​fi​
            
        - •
            
            DE2: ∑j∈/ICT​∑i∈ICT​vj​bj,i​fi​
            
        - •
            
            DE3: ∑j∈/ICT​∑i∈ICT​ci,j​，其中 ci,j​=(zi,j​/zj​)⋅vj​
            
        - •
            
            DCT: 双倍计算项调整。
            
        
    
- •
    
    **符号含义**：vj​为部门j的增加值系数，bj,i​为Leontief逆矩阵元素，zi,j​为中间投入。
    
- •
    
    **方法假设**：数字输入的价值贡献至少等于其投入份额，基于实证证据。
    
- •
    
    **选择理由**：I-O框架能捕捉部门间关联和溢出，适用于数字经济测量。
    

### 2.3 实现步骤（原子级分解）

**步骤1: 数据加载和预处理**

- •
    
    **步骤目的**：准备清洁数据用于I-O分析。
    
- •
    
    **输入数据**：原始ICIO和WDI数据。
    
- •
    
    **具体操作**：读取ICIO表，映射部门到ICT分类（表3）；应用PPP转换；计算增加值系数v_j。
    
- •
    
    **核心变量测算**：增加值系数v_j = 部门增加值 / 部门总产出。数据来源ICIO表，计算步骤为除法，示例：如中国计算机部门增加值除以总产出。
    
- •
    
    **参数设置**：无额外参数。
    
- •
    
    **中间输出**：清洁的部门级数据矩阵。
    
- •
    
    **验证检查**：检查部门加总与GDP一致。
    

**步骤2: 构建Leontief逆矩阵**

- •
    
    **步骤目的**：计算需求驱动的产出乘数。
    
- •
    
    **输入数据**：清洁的中间投入矩阵A。
    
- •
    
    **具体操作**：计算技术系数矩阵A（a_{i,j} = z_{i,j} / x_j）；计算Leontief逆矩阵B = (I - A)^{-1}。
    
- •
    
    **核心变量测算**：技术系数a_{i,j}基于中间投入z_{i,j}和总产出x_j。计算步骤为元素级除法。
    
- •
    
    **参数设置**：矩阵运算精度要求高，使用数值稳定算法。
    
- •
    
    **中间输出**：Leontief逆矩阵B。
    
- •
    
    **验证检查**：验证B矩阵性质，如非负性和收敛性。
    

**步骤3: 计算数字经济成分**

- •
    
    **步骤目的**：分解DE1, DE2, DE3。
    
- •
    
    **输入数据**：B矩阵、增加值系数、最终需求。
    
- •
    
    **具体操作**：根据公式6-13计算各成分；求和并调整DCT。
    
- •
    
    **核心变量测算**：DE1a、DE1b、DE2、DE3的求和计算。示例：DE1a为ICT部门内部交易的价值加总。
    
- •
    
    **参数设置**：无。
    
- •
    
    **中间输出**：各成分价值量。
    
- •
    
    **验证检查**：总DE与GDP份额合理（如10-20%）。
    

**步骤4: 计算比较指标**

- •
    
    **步骤目的**：生成相对水平和追赶速度。
    
- •
    
    **输入数据**：中美数字经济成分。
    
- •
    
    **具体操作**：相对水平R_t = X_{t,cur}^{CN} / X_{t,cur}^{US} * 100；追赶速度CUP = (X_{t1,const}^{CN} / X_{t0,const}^{CN}) / (X_{t1,const}^{US} / X_{t0,const}^{US})。
    
- •
    
    **核心变量测算**：增长率比值的计算。数据来源为常数价序列。
    
- •
    
    **参数设置**：时间区间1995-2020。
    
- •
    
    **中间输出**：指标时间序列。
    
- •
    
    **验证检查**：指标值在合理范围（如CUP>1表示追赶）。

# 研究-定题

基于AI的中美数字经济演变预测与政策影响分析

## 研究内容

### 1. 研究目的与意义

- •
    
    **现实背景**：数字经济驱动全球竞争，中美路径分歧需动态分析。
    
- •
    
    **政策背景**：各国数字战略需科学评估工具。
    
- •
    
    **意义**：AI优化提升预测精度，为政策提供数据驱动洞察，理论贡献于数字经济学和方法论创新。
    

### 2. 研究问题与假设

- •
    
    **核心问题**：AI方法如何增强中美数字经济演变的预测和政策效应评估？
    
- •
    
    **假设**：AI模型能显著降低预测误差，并量化政策干预的因果效应。
    

### 3. 研究方法与技术路线

- •
    
    **继承部分**：沿用I-O框架和数字经济分解。
    
- •
    
    **AI介入环节**：使用LSTM预测成分趋势；BERT分析政策文本；机器学习动态化技术系数。
    
- •
    
    **数据需求**：扩展1995-2020年数据，加入政策文档和宏观指标。
    
- •
    
    **评估指标**：预测误差（RMSE）、政策关联度（相关性）。
    

### 4. 预期结果与创新点

- •
    
    **主要结论**：AI模型提升预测精度20%以上；识别关键政策杠杆。
    
- •
    
    **创新点**：方法上融合AI与I-O分析；应用上提供动态政策模拟工具。
    
- •
    
    **局限**：数据依赖性强；对策：敏感性分析。
    

## 摘要

本研究基于1995-2020年投入-产出数据，引入人工智能方法优化中美数字经济的演变分析。通过机器学习动态化技术系数、深度学习预测趋势、自然语言处理量化政策影响，构建综合预测模型。结果表明，AI方法显著提升预测准确性，并揭示政策干预的非线性效应。理论上，推进数字经济学方法论创新；实践上，为中美数字战略提供动态评估工具，支持政策制定者设计精准干预措施。
